# Карта файлов проекта

Обновлять этот документ при добавлении, переименовании или удалении исходных файлов и сцен.

| Путь | Назначение |
|---|---|
| `project.godot` | Настройки Godot, стартовая сцена, рендерер и autoload `GameState`. |
| `AGENTS.md` | Инструкции и навигация для Codex и других агентов. |
| `docs/game_concept.md` | Утверждённый игровой концепт и границы MVP. |
| `docs/assets.md` | Происхождение и лицензии сторонних ассетов. |
| `docs/technical/architecture.md` | Схема сцен, состояния и ответственности компонентов. |
| `docs/technical/debugging.md` | Формат отладочного журнала и инструкция по его использованию. |
| `docs/technical/file_map.md` | Эта карта файлов. |
| `scenes/main_menu.tscn` | Главное меню серого прототипа. |
| `scenes/level_01.tscn` | Фиксированная трасса с крышами, стеной, тремя лужами урона, предметами и финишем. |
| `scripts/main_menu.gd` | Переход в трассу и выход из приложения. |
| `scripts/player.gd` | Автоматический бег, прыжок, здоровье и мигание в период неуязвимости. |
| `scripts/player_animation.gd` | Переключение спрайтовых листов кота между бегом, прыжком и уроном. |
| `scripts/collectible.gd` | Общая логика подбора писем и рыбок. |
| `scripts/level_01.gd` | Таймер, пауза, урон от луж, предметы, подсказка, падение, финиш и экраны результата. |
| `scripts/game_state.gd` | Локальное сохранение и форматирование лучшего времени. |
| `scripts/debug_log.gd` | Глобальный отладочный журнал, включаемый и выключаемый клавишей F3. |
| `assets/characters/cat/free_cat_2d_pixel_art/` | Локальная копия пака кота и его лицензия. |
