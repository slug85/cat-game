# Карта файлов проекта

Обновлять этот документ при добавлении, переименовании или удалении исходных файлов и сцен.

| Путь | Назначение |
|---|---|
| `project.godot` | Настройки Godot, стартовая сцена, рендерер и autoload `GameState`. |
| `AGENTS.md` | Инструкции и навигация для Codex и других агентов. |
| `docs/game_concept.md` | Утверждённый игровой концепт и границы MVP. |
| `docs/assets.md` | Происхождение и лицензии сторонних ассетов. |
| `docs/technical/architecture.md` | Схема сцен, состояния и ответственности компонентов. |
| `docs/technical/debugging.md` | Формат отладочного журнала и инструкция по его использованию. |
| `docs/technical/file_map.md` | Эта карта файлов. |
| `scenes/main_menu.tscn` | Главное меню серого прототипа. |
| `scenes/world_map.tscn` | Экран карты с доступными маршрутами 01, 02A и 02B. |
| `scenes/level_01.tscn` | Базовая трасса «Ночные крыши» с фасадами, опасными люками, предметами и воротами финиша. |
| `scenes/level_02_old_quarter.tscn` | Наследуемая ветвь «Старый квартал» с каменными верхними платформами. |
| `scenes/level_03_neon_bypass.tscn` | Наследуемая ветвь «Неоновый обход» с неоновыми верхними платформами. |
| `scripts/main_menu.gd` | Переход в трассу и выход из приложения. |
| `scripts/player.gd` | Автоматический бег, прыжок, здоровье, мигание в период неуязвимости и единичный сигнал упора в стену. |
| `scripts/player_animation.gd` | Переключение спрайтовых листов кота между бегом, прыжком и уроном. |
| `scripts/collectible.gd` | Общая логика подбора писем и рыбок. |
| `scripts/letter_animation.gd` | Трёхкадровая пульсация писем без изменения коллайдеров. |
| `scripts/central_city_art.gd` | Декоративная отрисовка фасадов крыш, Старого квартала, анимированного пара и ворот доставки. |
| `scripts/fish_animation.gd` | Покадровая анимация и покачивание рыбы-подборки. |
| `scripts/level_01.gd` | Таймер, пауза, урон от вентиляционных люков, предметы, звук событий, подсказка, падение, финиш и экраны результата. |
| `scripts/branch_level.gd` | Перестройка базового уровня для ветвей карты: предметы, опасности и верхние коллизионные платформы. |
| `scripts/branch_route_art.gd` | Отрисовка каменного и неонового вариантов ветвей. |
| `scripts/world_map.gd` | Открытие трасс с карты и возврат в главное меню. |
| `scripts/game_state.gd` | Локальное сохранение и форматирование лучшего времени. |
| `scripts/debug_log.gd` | Глобальный отладочный журнал, включаемый и выключаемый клавишей F3. |
| `assets/characters/cat/free_cat_2d_pixel_art/` | Локальная копия пака кота и его лицензия. |
| `assets/environment/central_city/` | PNG-листы города Central City и описание источника. |
| `assets/environment/fort_of_illusion/` | CC0-тайлы и пропсы Fort of Illusion для участка Старого квартала. |
| `assets/collectibles/animated_fish/` | Исходный GIF и шесть PNG-кадров CC0-анимации рыбы. |
| `assets/audio/music/cybertracks_vol_1/` | Временная музыка первого уровня с условиями кредита в `docs/assets.md`. |
| `assets/audio/sfx/jdsherbert_ultimate_ui/` | Выбранные OGG-эффекты и оригинальная лицензия UI-пака. |
