# Инструкции для Codex

Этот файл — точка входа для агентов Codex, работающих с проектом.

## Проект

- Игра: «Экспресс “Рыжий хвост”».
- Движок: Godot 4.x, язык: GDScript.
- Текущая цель: серый вертикальный срез (меню, автоматический бег, прыжок, трасса, финиш и падение).
- Игровой концепт и границы MVP: [`docs/game_concept.md`](docs/game_concept.md).

## Правила изменений

- Перед существенными изменениями сверяйся с `docs/game_concept.md`.
- Не добавляй в MVP функции из раздела «Не входит в MVP» без явного решения пользователя.
- Для каждого нового исходного файла или сцены обновляй `docs/technical/file_map.md`.
- Если изменение затрагивает взаимодействие сцен или состояние игры, обновляй `docs/technical/architecture.md`.
- Служебные папки `.godot/` и `.idea/` не коммитить.

## Проверка

- После изменения GDScript запускать проверку парсинга Godot в headless-режиме, если локальный исполняемый файл Godot доступен.
- Не менять ассеты, настройки экспорта или платформенные интеграции без отдельной задачи.
